Proyectos grupales

Libreria de comics.

El objetivo de esta aplicación es que el cliente tenga una librería donde pueda guardar y revisar sus libros, en nuestro caso, nuestra librería constaba de una de comics, las tecnologías se basaron en Handlebars con CSS y JS vanilla, usando como base de datos MongoDB.

Rama de alquiler.

En este proyecto simplemente nuestro grupo se encargó de desarrollar la parte de alquiler de una página web mayor de compra/venta/alquiler de vehículos, usando handlebars, CSS y JS.

Pagina conjunta de compra/venta/alquiler de vehiculos.

Como resultado de la rama de alquiler, se hizo un proceso de implementación de las diferentes ramas delos diferentes equipos, usando en todas las mismas tecnologías, Usando MongoDB para la base de datos.

App para peregrinos del camino de santiago.

La aplicación busca dar una ayuda tanto a peregrinos cono a los dueños de los albergues, mostrando los servicios que ofrecen y poniendo en comunicación a ambas partes, en este caso usamos VUE, y en el resto CSS y JS, con MongoDB como base de datos.
